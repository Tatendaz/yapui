# 🗣️ YapUI

> **Yap at your UI. Watch Claude rebuild it live.**

A [Claude Code](https://claude.com/claude-code) skill for live HTML preview with a two-way feedback loop — preview any page, then **talk · point · record · screenshot · type** your changes and watch Claude apply them live, no trips back to the terminal.

🚧 The full skill is landing via an open pull request (last step: the hero demo GIF). See the **Pull requests** tab.
